# CRM_Windykacja

CRM for debt collecition company

Main functionalities:
- add/delete users
- create new cases
- add/edit/delete new debts
- add/edit/delete debtors, emails, telephones and adresses.
- add/edit/delete action codes and action results
- sending SMS


Technologies used
- Java
- Spring Security
- Spring Data
- Spring Core 
- Hibernate
- MySQL
- JSP
- SMSAPI
